## 📊 Project Overview

The **Tech Workforce Intelligence Dashboard** analyzes global layoffs and hiring trends in the technology sector from **2020 to 2024**.

This project integrates **data analytics, machine learning, and business intelligence dashboards** to provide insights into workforce disruptions across industries and regions.

The system combines **historical layoff data with a Random Forest Machine Learning model** to classify companies into different **layoff risk tiers**.

---

## 🎯 Objectives

✔ Analyze global tech layoffs across industries  
✔ Identify workforce disruption patterns  
✔ Build an interactive **Power BI dashboard**  
✔ Predict **layoff risk levels using machine learning**

## 🧠 Machine Learning Model

Algorithm Used:

Why Random Forest?

- Handles imbalanced datasets
- Reduces overfitting
- Provides feature importance ranking

## 🔎 Key Insights

- **2023 recorded the highest layoffs globally**
- **USA had the largest layoff numbers**
- Retail and Consumer industries were most affected
- Public companies accounted for the majority of layoffs
